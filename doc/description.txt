Web ShioriはWebにしおりをはさめるアプリです。WebブラウジングのUXを改善します。

Web Shioriは以下の3つの機能を持っています。

1. 複数端末間でWebサイトや動画、PDFをどこまで視聴したか同期できる
    具体的には、Webページのスクロール位置・動画の再生位置・PDFのページ数を保存し、読んでいた/視聴していた位置で再度閲覧できます。
2. Webサイトや動画の面白いシーンや名言を保存していつでもすぐに見れる
3. Webサイトや動画の特定のテキストやシーンをかんたんに共有できる

iOSアプリ(https://apps.apple.com/jp/app/shiori-web-for-safari/id1480539987)と連携すれば、複数デバイス・複数ブラウザ間でブックマークを共有することができます。

Web Shioriは完全にオープンソースのアプリケーションです。
https://github.com/web-shiori

【注意事項】
※各動画サイト・webサイトの仕様上、状態を保存できないサイトもございます。
※各動画サイト・webサイトの仕様上、テキストを共有できないサイトもございます。
